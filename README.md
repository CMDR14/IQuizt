# Build
- Ubuntu:
  - Get packages:
    > sudo apt install -y cmake  
    > sudo apt install -y make  
    > sudo apt install -y gcc  
    > sudo apt install -y qt5-default  
  - Run ubuntu_build.sh
  - Tested on Ubuntu 20.04.2 LTS
